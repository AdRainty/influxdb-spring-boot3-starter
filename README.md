# influxdb-spring-boot3-starter

Use influxdb like mybatis & mybatis plus

## Base environment

- Java 17
- Springboot 3

## Usage

pom.xml
~~~xml
<dependency>
    <groupId>io.adrainty</groupId>
    <artifactId>influxdb-spring-boot3-starter</artifactId>
    <version>1.0-SNAPSHOT</version>
</dependency>
~~~
